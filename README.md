# Лабораторные работы по SQL


### Лабораторная работа №1: Основы SQL (SELECT, WHERE, ORDER BY, CRUD)
В ходе первой работы были освоены базовые операции SQL: выборка данных с фильтрацией (`WHERE`), сортировка (`ORDER BY`), а также полный цикл CRUD-операций — создание таблицы (`CREATE TABLE AS`), обновление (`UPDATE`) и удаление записей (`DELETE`). Получены навыки работы с условиями `BETWEEN` и базовой структурой SQL-запросов.
https://github.com/BratchenkoAA/SQL-2026/tree/main/SQL-lb/SQL-lb_01

### Лабораторная работа №2: Соединения таблиц и преобразование данных
Вторая работа была посвящена работе с несколькими таблицами: изучены различные типы соединений (`INNER JOIN`, `LEFT JOIN`), применение подзапросов (`Subquery`) для решения сложных задач поиска, а также использование условной логики с конструкцией `CASE WHEN` для классификации данных и создания вычисляемых столбцов.
https://github.com/BratchenkoAA/SQL-2026/tree/main/SQL-lb/SQL-lb_02

### Лабораторная работа №3: Агрегация данных
В третьей работе освоены методы агрегирования данных: применение агрегатных функций (`MIN`, `MAX`, `AVG`, `COUNT`, `SUM`), группировка данных с помощью `GROUP BY`, а также фильтрация сгруппированных результатов через `HAVING`. Полученные навыки позволяют проводить статистический анализ и формировать сводные отчеты по данным.
https://github.com/BratchenkoAA/SQL-2026/tree/main/SQL-lb/SQL-lb_03
